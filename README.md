# AI Interview / Presentation Tone Checker

This project analyzes interview or presentation responses and detects the speaker's tone using NLP sentiment analysis.

## Features
- Tone detection (Confident, Neutral, Nervous)
- Polarity and subjectivity analysis
- Confidence score visualization
- Feedback and suggestions

## Technologies Used
- Python
- Streamlit
- TextBlob
- NLP

## How to Run
1. Install Python
2. Install dependencies:
   pip install streamlit textblob
3. Run the app:
   streamlit run app.py
